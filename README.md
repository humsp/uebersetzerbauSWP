# Twee2Z
A compiler for converting **Twee source code** (http://twinery.org/) into **Z-code** (https://en.wikipedia.org/wiki/Z-machine).
The resulting Z-code can be executed with Z-machines like Frotz (http://frotz.sourceforge.net/).
